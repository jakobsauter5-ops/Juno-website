# JUNO St. Moritz Website Sample

Dark premium one-page front-end sample for JUNO St. Moritz.

## Run

Open `index.html` in a browser, or serve the folder with any static server.

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Notes

- This is a design sample only, not a production website.
- Events are demo content and are generated from `scripts/main.js`.
- Ticket buttons open a demo modal instead of a real checkout flow.
- `assets/juno-logo-crop.png` is used as the logo in the header and hero.
- `assets/demo_video_sito.mp4` is used as an atmospheric local hero background.
